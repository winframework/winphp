**Framework em PHP baseado em MVC**

- Responsável por incluir as páginas de acordo com a URL e gerenciar a estrutura MVC.
- Assim você não precisa se preocupar com a estrutura de URL, páginas, títulos, etc...

### Requisitos

| PHP 5.6 +
